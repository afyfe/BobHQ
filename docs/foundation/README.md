# AskBob Foundation

AskBob is being built as a grounded operational intelligence platform for SMEs.

Core principles:

- Useful over impressive
- Calm execution over rushed hype
- Operational visibility matters
- Simplicity matters
- Build for real business problems

MORSE:

- Modular
- Opinionated
- Role-based
- Simple
- Epic

NAIB:

Not Another AI Buzzword Platform

AskBob should remain practical, explainable, operationally useful and human.
