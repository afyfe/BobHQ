# Onboarding Process

Synthetic SharePoint process note.

Last reviewed: 2025-09-18
Owner: Operations

## Current Steps

1. Sales uploads signed agreement.
2. Finance checks PO reference.
3. Operations assigns install owner.
4. Engineer confirms access requirements.
5. Customer receives installation date.
6. Training checklist is completed after install.

## Known Issues

- This document still says finance checks happen after install scheduling, but operations now says finance hold should be cleared first.
- Owner field is often left blank on the install board.
- Pricing exceptions are not always copied into CRM.
