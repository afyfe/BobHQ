# Onboarding Checklist

Synthetic internal checklist.

- Signed agreement stored in SharePoint
- PO reference confirmed or marked not required
- Finance hold checked in Sage
- Install owner assigned
- Engineer assigned
- Site contact confirmed
- Access hours confirmed
- Training owner assigned
- Customer receives install date
- Post-install issue window monitored for 7 days

## Note

The checklist says training owner must be assigned, but the install board does not currently have a training owner field.
