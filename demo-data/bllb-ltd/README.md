# BLLB LTD

Synthetic SME operational dataset for AskBob demos and discovery workflows.

BLLB LTD is a fictional company used to simulate:

- operational fragmentation
- disconnected systems
- document chaos
- tribal knowledge
- CRM/finance inconsistencies
- operational discovery scenarios

## Important

This dataset must remain fully synthetic.
No real customer or Aquacool operational data should be used.
